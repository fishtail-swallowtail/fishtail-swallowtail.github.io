# 当说者被说的时候：比较叙述学导论（赵毅衡）读书札记

‘美国文论家诺尔曼·弗利德曼则建议分成八类：全知叙述有作者干预；全知叙述无作者干预；第一人称叙述观察者式；第一人称叙述主角式；有限视角全知式，复式；有限视角全知式，单式；纯客观叙述戏剧式；纯客观叙述记录式。’

【】念头：颇有参考价值，可供写作方式的参考。

# 广义叙述学 (赵毅衡) 读书札记

【】念头：以前写大学参考书的一些老先生，无论理工农医文史哲，抽屁股算是轻的，最好集中起来自己朗诵自己的大著100遍，可读性极差，还要害学生们记忆其中的内容，也不知道这些所谓专家，生前能否有机会意识到这个。窃以为比较不错的参考书，应该是作者先吃透嚼碎，通顺通俗表达，并加入自己当初的学习技巧或经验，一并传之，方能有利于后辈徒弟学习，才能称得起传道授艺解惑这一名号。


‘有学者甚至认为人的生存必需序列，应当是“食一述一性一住”（food-telling-sex-shelter）。因为“许多人没有性，没有住所，也活了下来，但几乎没有人能在沉默中生存”。’

【】念头：如果这个观点正确，近几年来，十五先生常数日不发一言，大概率是已经修炼到一定境界了。

# 符号逻辑讲义 (徐明) 读书札记
【】念头：对文科生而言，书里数学符号偏多，他们会头大；对理科生而言，书里数学内容没那么难，也许会好奇为什么还会有这样的学科，可能最大的作用，是能够让文科同学们多一些逻辑思考。
